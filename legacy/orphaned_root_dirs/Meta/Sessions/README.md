## Usage

After each significant session:
1. Create a session summary in this folder
2. Include: Date, Agent(s), Tasks completed, Decisions made, Files created
3. Reference specific commits or task IDs
4. Useful for:
   - Understanding project evolution
   - Onboarding new agents
   - Debugging decisions
   - Learning from past work
