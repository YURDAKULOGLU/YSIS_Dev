# Chainlit Framework Summary

## Key Features
- Fast Build: Create AI interfaces in minutes.
- Multi-Step Reasoning: Visualize intermediate thoughts and actions.
- Data Persistence: Automatic conversation history.
- Multi-Platform: Web, Slack, Discord support.

## API Concepts
- @cl.on_chat_start: Initialize session.
- @cl.on_message: Handle user input.
- cl.Step: Group actions into steps for visualization.
- cl.Message: Send final response.
