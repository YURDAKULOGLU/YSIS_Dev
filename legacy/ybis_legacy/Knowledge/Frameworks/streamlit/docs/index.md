# Streamlit documentation

[Streamlit](https://www.streamlit.io) is an open-source Python framework for data scientists and AI/ML engineers to deliver dynamic data apps with only a few lines of code. Build and deploy powerful data apps in minutes. Let's get started!

## What's new

[*schedule*

#### Date & time input

Set date and time in a single widget with the new `st.datetime_input`.](/develop/api-reference/widgets/st.datetime_input?utm_source=streamlit)[*download*

#### Deferred downloads

Pass a callable to `st.download_button` for on-click download generation without blocking your script.](/develop/api-reference/widgets/st.download_button?utm_source=streamlit)[*mic*

#### Audio chat input

`st.chat_input` now supports audio input for voice-enabled chat interactions.](/develop/api-reference/chat/st.chat_input?utm_source=streamlit)[*keyboard*

#### Keyboard shortcuts for buttons

Configure keyboard shortcuts for buttons to improve user experience and accessibility.](/develop/api-reference/widgets/st.button?utm_source=streamlit)[*link*

#### Query parameters for navigation

Pass query parameters to `st.switch_page` and `st.page_link` for enhanced navigation.](/develop/api-reference/navigation/st.switch_page?utm_source=streamlit)[*code*

#### JavaScript execution

Execute JavaScript in `st.html` with the new `unsafe_allow_javascript` parameter.](/develop/api-reference/text/st.html?utm_source=streamlit)
