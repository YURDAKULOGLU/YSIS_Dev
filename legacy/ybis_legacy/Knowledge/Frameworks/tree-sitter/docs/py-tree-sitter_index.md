# py-tree-sitter[#](#py-tree-sitter "Link to this heading")

Python bindings to the Tree-sitter parsing library.

## Constants[#](#constants "Link to this heading")

tree\_sitter.LANGUAGE\_VERSION *= 15*[#](#tree_sitter.LANGUAGE_VERSION "Link to this definition")
:   The latest ABI version that is supported by the current version of the library.

    Note

    When a [`Language`](classes/tree_sitter.Language.html#tree_sitter.Language "tree_sitter.Language") is generated by the Tree-sitter CLI, it is assigned
    an ABI version number that corresponds to the current CLI version.
    The Tree-sitter library is generally backwards-compatible with languages
    generated using older CLI versions, but is not forwards-compatible.

tree\_sitter.MIN\_COMPATIBLE\_LANGUAGE\_VERSION *= 13*[#](#tree_sitter.MIN_COMPATIBLE_LANGUAGE_VERSION "Link to this definition")
:   The earliest ABI version that is supported by the current version of the library.

tree\_sitter.\_\_version\_\_ *= '0.25.2'*[#](#tree_sitter.__version__ "Link to this definition")
:   The version of the tree-sitter package.
