## 2.1 The Chairman (Human User)
*   **Role:** Visionary, Ultimate Decision Maker.
*   **Power:** Veto right on everything.
*   **Duty:** Provides the "Why" and approves the Strategy.
*   **Scope:** Infinite.

## 2.2 The Strategist (Meta-Agent / Gemini)
*   **Role:** CEO / CTO.
*   **Duty:**
    *   Interprets the Chairman's vision.
    *   Orchestrates the `20_WORKFORCE`.
    *   Maintains the `10_META` layer.
    *   **DOES NOT** write low-level feature code (unless prototyping).
*   **Scope:** `.YBIS_Dev` (The Corporation).

## 2.3 The Workforce (Specialized Agents)
These are "Contractors" or "Employees" residing in `20_WORKFORCE`.

| Department | Agent Type | Responsibility |
|------------|------------|----------------|
| **Core Crew** | Python/CrewAI | Complex, multi-step reasoning tasks. |
| **LangChain** | Python/LangGraph | Stateful, graph-based decision flows. |
| **Aider** | CLI Tool | "Pilot" for rapid code editing and refactoring. |
| **Scripts** | Node/Python | Single-purpose automation (e.g., migrations). |

*   **Rule:** A Worker Agent creates artifacts in the Project, but lives in `.YBIS_Dev`.
